ASP.NET Web API C# Samples for Visual Studio 2012

Here you will find samples using ASP.NET Web API in various ways for Visual Studio 2012 written in C#.

  • ContentControllerSample: ASP.NET hosted sample illustrating how to read and write request and response entities asynchronously using streams (both VS 2010 and VS 2012)
  • CustomAssemblyResolverSample: Custom Assembly Resolver sample showing how to load ApiControllers dynamically from a controller DLL (VS 2012)
  • FileUploadSample: File upload sample showing how to upload a file asynchronously using MIME multipart to an ApiController. The client side is HttpClient (VS 2012)
  • HostedBatchSample: Hosted Batching Scenario Sample showing how to implmenent batching scenario under web-host (VS2012)
  • JsonUploadSample: Json upload sample showing how to upload JSON to an ApiController using HttpClient (VS 2012)
  • MashupSample: Mashup sample showing how to orchestrate two asynchronous HTTP requests from within an ApiController action (VS 2012)
  • MongoSample: MongoDB sample showing how to use MongoDB for persistence using a repository pattern (VS 2012)
  • ResponseEntityProcessorSample: HTTP Response Entity processor sample showing how to copy a response entity (body) to a local file and perform additional processing on that file asynchronously (VS 2012)
  • UploadXDocumentSample: Upload XDocument to an ApiController using a PushStreamContent (VS 2012)
