HostedBatchSample

This sample is an ASP.NET Web API C# Sample for Visual Studio 2012. It shows how to implement batching scenario under web-host.

To run the sample, build the project and deploy HostedBatching server. The default deployment configuration is IIS Express. And then compile and run the HostedBatching.TestClient. The client makes request to http://localhost:29135, which is the default Project Url if you don't change the server configuration. If you decide to use someting other than IIS Express or use different url, you need to update the test client.
