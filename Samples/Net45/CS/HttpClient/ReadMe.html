HttpClient C# Samples for Visual Studio 2012

Here you will find samples using HttpClient in various ways for Visual Studio 2012 written in C#.

  • BingTranslateSample: Bing translator sample showing how to use HttpClient to talk to the Bing translator API including how to do OAuth
  • GoogleMapsSample: Google map sample showing how to use the Google Map API
  • TwitterSample: Twitter client sample showing how to retrieve data from twitter using OAuth
  • WorldBankSample: World Bank sample showing how to retrieve data from the World Bank data site using JSON.NET to parse the result
